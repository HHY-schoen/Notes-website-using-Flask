## note website
provide a online notes app which can create a new user's account, log in / out, associate info with specific user, store info to database...

Project include :  
- Create a Flask App
- Create Routes/Views
- Jinja Templating Language & HTML Templates
- Sign up, log page
- Message Flashing
- Set up SQLAlchemy
- Database Models 
- Foreign Key Relationships
